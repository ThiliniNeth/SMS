# 📚 Student Management System (SMS)

A web-based Student Management System developed using PHP and MySQL to manage students, teachers, admins, assignments, and academic activities efficiently.


## 🚀 Features

* Multi-user Authentication (Admin / Teacher / Student / Academic Staff)
* Student Management (Add, Update, Delete, Search)
* Teacher Management
* Admin & Academic Staff Management
* Assignment Management (Upload, Download, Submit)
* Lesson Notes Management
* Marks & Results Management
* Search Functionality for all user types
* File Upload & Download Support
* Responsive UI using Bootstrap



## 🛠️ Technologies Used

* Frontend:

  * HTML
  * CSS
  * JavaScript
  * Bootstrap

* Backend:

  * PHP

* Database:

  * MySQL

* Tools:

  * XAMPP
  * MySQL Workbench



## 🔑 System Modules

* Admin Dashboard
* Teacher Dashboard
* Student Dashboard
* Academic Staff Dashboard

Each role has different access and functionalities.
